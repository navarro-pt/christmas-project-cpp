# Christmas Console Suite 🎄

A festive C++ terminal application developed as a school project. It features high-quality ASCII art, dynamic console animations, and an interactive guessing game.

## 🚀 Features
- **Dynamic Christmas Tree:** An animated ASCII tree with "blinking" lights using console color manipulation.
- **ASCII Santa Claus:** A detailed rendering of Santa Claus in the terminal.
- **The Gift Game:** A multi-difficulty guessing game (Easy, Normal, Insane) with randomized logic.
- **Custom UI:** Uses `windows.h` for precise cursor positioning and text attributes.

## 🛠️ Tech Stack
- **Language:** C++
- **Platform:** Windows (uses `windows.h` API)
- **Concepts used:** Switch-case menus, `do-while` loops, random number generation (`srand`), and console manipulation.

## 📖 How to Run
1. Clone this repository.
2. Open the `.cpp` file in a Windows-compatible compiler (e.g., Dev-C++, Visual Studio, or MinGW).
3. Compile and run the executable.
*Note: This application is designed for the Windows Console.*
